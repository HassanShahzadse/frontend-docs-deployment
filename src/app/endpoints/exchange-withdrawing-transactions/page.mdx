# Exchange Withdrawing Transactions

The **Exchange Withdrawing Transactions** metric tracks the total number of outflow transactions from exchanges. {{ className: 'lead' }}

This metric provides insights into withdrawal transaction volume, helping identify periods of heightened activity, batch processing patterns, and overall exchange operational health.

---

## Endpoint

`GET /exchange-withdrawing-transactions`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

## Understanding the metric

<Row>
  <Col>

<Properties>
  <Property name="transactions_count_outflow" type="integer">
    The total number of withdrawal transactions from the exchange during the
    specified time window.
  </Property>
</Properties>

### Interpretation

**Rising transaction count** indicates increased withdrawal activity and may signal heightened user concern, market volatility responses, or normal operational scaling.

**Declining transaction count** suggests reduced withdrawal activity, which could indicate stable market conditions or users keeping funds on exchange for trading.

**Sudden spikes** may reveal potential panic withdrawals, security concerns, or batch processing events.

**Regular patterns** help identify exchange operational schedules, such as scheduled batch withdrawal processing times.

### Use Cases

High transaction count with low address count (from the Withdrawing Addresses endpoint) suggests batch processing or few large withdrawals. Sudden drops may indicate technical issues or withdrawal suspensions. Spikes often correlate with negative news or market uncertainty, making this useful for sentiment analysis.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-withdrawing-transactions', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-withdrawing-transactions?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      transactions_count_outflow: d.transactions_count_outflow,
    }))
    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
