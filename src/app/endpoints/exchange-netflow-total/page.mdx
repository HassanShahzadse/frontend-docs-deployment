# Exchange Netflow (Total)

The **Exchange Netflow (Total)** metric measures the net amount of Bitcoin entering or leaving exchange wallets — calculated as inflow minus outflow. {{ className: 'lead' }}

A positive netflow suggests BTC is moving **into** exchanges (potentially indicating sell pressure), while a negative netflow suggests BTC is **leaving** exchanges (potentially accumulation or HODLing behavior).

---

## Endpoint

`GET /exchange-netflow-total`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

```bash {title="Requesting Exchange Netflow (Total) data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-netflow-total" \
  -H "x-api-key: YOUR_API_KEY" \
  -d exchange=binance \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

<Properties>
  <Property name="netflow_total" type="number">
    Net BTC flow to exchanges = inflow − outflow. Positive values indicate BTC
    entering exchanges; negative values indicate BTC leaving exchanges.
  </Property>
</Properties>

### Interpretation

- **Positive Netflow** → More BTC entering exchanges → potential **sell-side pressure**.
- **Negative Netflow** → More BTC leaving exchanges → potential **accumulation** or **HODL** behavior.

Tracking netflow across major exchanges (e.g., Binance, Coinbase, Kraken) can provide a high-level overview of market sentiment and liquidity shifts.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-netflow-total', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-netflow-total?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      netflow_total: d.netflow_total,
    }))
    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
