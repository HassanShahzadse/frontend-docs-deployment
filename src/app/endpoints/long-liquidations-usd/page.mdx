# Long Liquidations USD

The **Long Liquidations USD** metric tracks the dollar value of forced market orders closing leveraged long positions. {{ className: 'lead' }}

This metric reveals the economic scale of overleveraged bullish positions being forcibly closed, providing a normalized view of liquidation severity across different price environments.

---

## Endpoint

`GET /long-liquidations-usd`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

## Understanding the metric

<Row>
  <Col>

<Properties>
  <Property name="long_liquidations_usd" type="number">
    The total dollar value of leveraged long positions forcibly closed due to
    insufficient margin, measured in USD.
  </Property>
</Properties>

### Interpretation

**High USD liquidations** reveal significant dollar-value destruction from overleveraged long positions, indicating the real economic impact of cascade events. Large USD values often correlate with major market corrections and panic selling.

**Liquidation spikes exceeding $100M+** typically mark significant capitulation events where massive amounts of leveraged capital are wiped out, often preceding market bottoms as the last overleveraged participants are cleared.

**Rising USD liquidations despite stable prices** suggest increasing leverage usage at current price levels, warning that the market is becoming more fragile and vulnerable to cascade events from small price moves.

**Low USD liquidations during bull markets** indicate healthy spot-driven rallies without excessive leverage, suggesting more sustainable price appreciation with lower cascade risk and correction potential.

### Use Cases

This metric provides crucial insights into the real economic scale of leverage unwinding events. USD-denominated values allow meaningful comparisons across different price regimes, showing whether current liquidations are historically significant or routine. Hundred-million-dollar liquidation events often mark major inflection points where overleveraged sentiment reaches extremes. Comparing USD liquidations with BTC-denominated values reveals whether high liquidation counts result from many small positions or few large whales being forced out. For risk assessment, tracking daily USD liquidation totals helps identify when leverage in the system reaches dangerous levels that could trigger systemic cascades. Historical analysis shows liquidation events exceeding several hundred million dollars typically clear out weak hands and create strong support levels. Monitoring liquidations across exchanges in USD terms reveals which platforms carry the most systemic risk from concentrated leverage.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/long-liquidations-usd', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const symbol = req.query.symbol || 'all_symbol'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `long-liquidations-usd?window=${window}&exchange=${exchange}&symbol=${symbol}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      long_liquidations_usd: d.long_liquidations_usd,
    }))
    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
