# Exchange Outflow (Total)

The **Exchange Outflow (Total)** metric represents the **total volume of BTC withdrawn from exchange wallets** over a selected time period. {{ className: 'lead' }}

High outflows often suggest accumulation or capital flight from exchanges, while low outflows may indicate reduced investor activity or stronger exchange retention.

---

## Endpoint

`GET /exchange-outflow-total`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

---

```bash {title="Requesting Exchange Outflow (Total) data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-outflow-total" \
  -H "x-api-key: YOUR_API_KEY" \
  -d exchange=binance \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

<Properties>

    Total amount of BTC withdrawn from exchange wallets during the selected

  </Property>
</Properties>

### Interpretation

- **High Total Outflow (↑)** → Indicates accumulation or capital withdrawal, usually **bullish** (investors moving BTC to cold storage).
- **Low Total Outflow (↓)** → Suggests fewer withdrawals, potentially **neutral** or **bearish** depending on inflow data.

Correlate with inflow and netflow metrics to better understand market behavior and liquidity changes.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-outflow-total', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-outflow-total?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      outflow_total: d.outflow_total,
    }))
    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
