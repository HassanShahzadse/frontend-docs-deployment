# UTXO Age Bands (USD)

The **UTXO Age Bands (USD)** metric distributes the active Bitcoin supply across age cohorts in dollar values, revealing the economic scale of holdings by different holder types. {{ className: 'lead' }}

This metric identifies the dollar-denominated value concentration across age groups, enabling analysis of the economic significance of short-term trader versus long-term holder positions at current market prices.

---

## Endpoint

`GET /utxo-age-bands-usd`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

### Interpretation

**High young band dollar values** indicate substantial economic value in recently-moved Bitcoin, revealing billions in fresh capital actively trading without establishing long-term positions, typical during high-volume periods.

**High old band dollar values** demonstrate massive economic value locked in unmoved Bitcoin, showing hundreds of billions in patient capital demonstrating conviction through years of inactivity, strengthening supply fundamentals.

**Growing old band dollar values** reveal increasing economic capital maturing into conviction, signaling successful accumulation where substantial value transitions from active trading to long-term holding, reducing available liquid supply.

**Declining old band dollar values** signal long-term holder distribution where substantial economic value from aged positions enters circulation, often marking major profit-taking events as veteran capital breaks its commitment.

### Use Cases

This metric provides dollar-normalized insight into the economic significance of different holder age cohorts, revealing the actual capital value at stake in each segment. High old band dollar values indicate massive economic capital has demonstrated conviction through sustained inactivity, with hundreds of billions locked in long-term positions representing the patient capital base. These dollar values contextualize the economic impact of potential distribution events, as movements from high-value old bands represent substantially more market-moving capital than equivalent BTC amounts from young bands. The metric also reveals how market capitalization changes affect holder economics, with rising prices inflating old band dollar values and demonstrating growing unrealized gains for veteran holders. Tracking old band dollar value growth identifies wealth accumulation in patient capital, as long-term holders benefit disproportionately from sustained appreciation. Comparing dollar values across age bands reveals capital concentration patterns, with disproportionate old band values indicating wealth has accumulated in conviction holders while balanced distributions suggest more democratic capital allocation. The metric proves valuable for assessing potential distribution pressure, as high old band dollar values indicate substantial profitable capital that could theoretically exit positions. During bull markets, tracking old band dollar values identifies when veteran holdings reach historically elevated levels that have preceded profit-taking events.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/utxo-age-bands-usd', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `utxo-age-bands-usd?window=${window}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      range_0d_1d_usd: d.range_0d_1d_usd,
      range_1d_1w_usd: d.range_1d_1w_usd,
      range_1w_1m_usd: d.range_1w_1m_usd,
      range_1m_3m_usd: d.range_1m_3m_usd,
      range_3m_6m_usd: d.range_3m_6m_usd,
      range_6m_12m_usd: d.range_6m_12m_usd,
      range_12m_18m_usd: d.range_12m_18m_usd,
      range_18m_2y_usd: d.range_18m_2y_usd,
      range_2y_3y_usd: d.range_2y_3y_usd,
      range_3y_5y_usd: d.range_3y_5y_usd,
      range_5y_7y_usd: d.range_5y_7y_usd,
      range_7y_10y_usd: d.range_7y_10y_usd,
      range_10y_inf_usd: d.range_10y_inf_usd,
    }))
    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
