# UTXO Value Bands (%)

The **UTXO Value Bands (%)** metric distributes the active Bitcoin supply across UTXO value size cohorts as percentages, revealing the relative wealth concentration. {{ className: 'lead' }}

This metric identifies the proportional distribution of supply by output size, enabling normalized analysis of whether wealth concentrates in numerous small retail outputs or fewer large whale positions.

---

## Endpoint

`GET /utxo-value-bands-percent`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

### Interpretation

**High small value percentages** indicate supply dominated by retail-sized outputs, revealing wealth disperses across many modest positions rather than concentrating in large discrete holdings, suggesting democratized distribution.

**High large value percentages** demonstrate supply dominated by whale-sized outputs, showing substantial wealth concentration in fewer but much larger positions held by sophisticated capital, indicating centralized distribution.

**Rising small value percentages** reveal supply dispersing into smaller outputs, signaling distribution or retail accumulation fragmenting holdings into more numerous modest positions, reducing whale dominance.

**Rising large value percentages** signal supply concentrating into whale outputs, indicating accumulation consolidates wealth into fewer but larger positions as sophisticated capital strengthens dominance over supply distribution.

### Use Cases

This metric provides normalized insight into wealth concentration patterns that enables meaningful comparison across different supply levels and market conditions. The percentage view removes the influence of absolute supply changes, focusing on the relative character of wealth distribution regardless of total Bitcoin in circulation. High large value percentages indicate wealth concentration where substantial supply consolidates in whale-sized outputs, suggesting sophisticated holders dominate and maintain significant positions. These concentrated distributions reveal markets where patient capital controls large supply proportions, creating potential for coordinated behavior as fewer entities hold more power. Conversely, high small value percentages indicate wealth dispersion where supply fragments across many retail-sized outputs, suggesting more democratized ownership where no single tier dominates supply control. The metric tracks wealth concentration shifts through percentage migration patterns, with growing large percentages confirming accumulation centralizes supply while growing small percentages indicate distribution democratizes holdings. Comparing percentage distributions across cycles reveals whether current wealth concentration demonstrates stronger or weaker centralization than historical precedent, contextualizing market structure risks. Rising large value percentages during bear markets confirm whale accumulation concentrates supply as sophisticated capital consolidates positions while retail capitulates. Declining large value percentages during bull markets warn of whale distribution as patient capital fragments large positions into smaller outputs for gradual profit-taking. The metric also identifies retail participation strength through small and mid-tier percentage growth, with increasing proportions indicating successful retail accumulation despite whale presence.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/utxo-value-bands-percent', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `utxo-value-bands-percent?window=${window}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      range_0_001_percent: d.range_0_001_percent,
      range_001_01_percent: d.range_001_01_percent,
      range_01_1_percent: d.range_01_1_percent,
      range_1_10_percent: d.range_1_10_percent,
      range_10_100_percent: d.range_10_100_percent,
      range_100_1k_percent: d.range_100_1k_percent,
      range_1k_10k_percent: d.range_1k_10k_percent,
      range_10k_inf_percent: d.range_10k_inf_percent,
    }))
    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
