# Exchange to Miner Flow (Mean)

The **Exchange to Miner Flow Mean** metric tracks the average size of token transfers from exchanges to mining pools. {{ className: 'lead' }}

This metric reveals miner purchasing behavior and accumulation patterns. Rising mean values often indicate larger miner acquisitions, suggesting bullish sentiment from those closest to network fundamentals.

---

## Endpoint

`GET /exchange-to-miner-flow-mean`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

```bash {title="Requesting Exchange to Miner Flow Mean data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-to-miner-flow-mean" \
  -H "x-api-key: YOUR_API_KEY" \
  -d from_exchange=binance \
  -d to_miner=f2pool \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

<Properties>
  <Property name="flow_mean" type="number">
    The average size of token transfers from the exchange to the mining pool
    during the specified time window.
  </Property>
</Properties>

### Interpretation

**Rising mean values** indicate larger average transfers to miners, often signaling increased miner accumulation or purchasing of tokens. This can be bullish as miners, who have intimate knowledge of network fundamentals and operational costs, are acquiring more tokens.

**Declining mean values** suggest smaller transfers, which may indicate miners taking smaller positions, operational purchasing for immediate needs, or reduced confidence in holding positions.

**Sudden spikes** typically reveal significant miner acquisitions, possibly indicating expectations of price appreciation or strategic accumulation by mining operations with strong balance sheets.

**Consistent patterns** help identify regular miner purchasing behaviors, seasonal trends tied to mining economics, or strategic accumulation cycles by major mining pools.

### Use Cases

This metric provides insights into miner sentiment, as miners are sophisticated market participants with direct exposure to production costs and network health. Rising mean values during price dips can signal that miners view current prices as undervalued relative to mining economics. When combined with Miner to Exchange flows, you can identify whether miners are in accumulation or distribution mode. Tracking specific mining pool behaviors can reveal which operations have the strongest balance sheets and most aggressive accumulation strategies.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-to-miner-flow-mean', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const fromExchange = req.query.from_exchange || 'all_exchange'
    const toMiner = req.query.to_miner || 'all_miner'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-to-miner-flow-mean?window=${window}&from_exchange=${fromExchange}&to_miner=${toMiner}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      flow_mean: d.flow_mean,
    }))
    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
