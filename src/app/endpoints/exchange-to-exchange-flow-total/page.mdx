# Exchange to Exchange Flow (Total)

The **Exchange to Exchange Flow Total** metric tracks the total volume of tokens transferred between exchanges. {{ className: 'lead' }}

This metric reveals inter-exchange liquidity movements, major arbitrage flows, and institutional rebalancing operations. High volumes indicate significant capital shifts between platforms.

---

## Endpoint

`GET /exchange-to-exchange-flow-total`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

```bash {title="Requesting Exchange to Exchange Flow Total data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-to-exchange-flow-total" \
  -H "x-api-key: YOUR_API_KEY" \
  -d from_exchange=binance \
  -d to_exchange=coinbase \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

<Properties>
  <Property name="flow_total" type="number">
    The total volume of tokens transferred from the source exchange to the
    destination exchange during the specified time window.
  </Property>
</Properties>

### Interpretation

**Rising total flow** indicates increasing liquidity movement between exchanges, often signaling arbitrage opportunities, platform migration, or institutional rebalancing operations as traders respond to price discrepancies.

**Declining total flow** suggests reduced inter-exchange activity, which may indicate stable pricing across platforms, reduced arbitrage opportunities, or shifts in trading patterns.

**Sudden spikes** typically reveal major arbitrage events, exchange listing announcements, or responses to significant price gaps between platforms that trigger large-scale rebalancing.

**Directional patterns** help identify which exchanges are gaining or losing liquidity. Consistent outflows from one exchange to another may signal declining confidence or better trading conditions elsewhere.

### Use Cases

This metric is essential for monitoring overall market liquidity distribution across exchanges. High total flows between specific exchange pairs often indicate active arbitrage corridors and pricing inefficiencies. When combined with the Flow Mean metric, you can distinguish between many small transfers versus few large institutional movements. Tracking flows from tier-1 exchanges like Binance to smaller platforms helps identify emerging exchange trends and potential listing impacts.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-to-exchange-flow-total', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const fromExchange = req.query.from_exchange || 'all_exchange'
    const toExchange = req.query.to_exchange || 'spot_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-to-exchange-flow-total?window=${window}&from_exchange=${fromExchange}&to_exchange=${toExchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      flow_total: d.flow_total,
    }))
    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
