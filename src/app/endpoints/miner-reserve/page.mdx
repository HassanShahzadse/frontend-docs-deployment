# Miner Reserve

The **Miner Reserve** metric tracks the total on-chain balance held by mining pool wallets. {{ className: 'lead' }}

This metric reveals how much Bitcoin miners are holding versus distributing, providing insights into miner sentiment, accumulation strategies, and potential future selling pressure from accumulated holdings.

---

## Endpoint

`GET /miner-reserve`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

## Understanding the metric

  <Col>

### Interpretation

**Rising miner reserves** indicate net accumulation where miners are holding more than they distribute, suggesting bullish sentiment from network operators who expect price appreciation and choose to build holdings rather than immediately liquidate.

**Declining miner reserves** reveal net distribution where miners are selling more than they accumulate from new production, indicating either bearish sentiment, profitability pressures requiring liquidation, or strategic profit-taking during favorable market conditions.

**All-time high reserves** demonstrate maximum miner confidence and accumulation, with operators choosing to hold rather than sell despite potentially favorable selling opportunities, suggesting strong conviction in future price appreciation.

**Sharp reserve drops** signal major distribution events where miners execute significant selling or treasury movements, potentially indicating capitulation, urgent liquidity needs, or strategic position exits that could pressure markets.

### Use Cases

This metric is fundamental for understanding miner sentiment and predicting future supply dynamics. Rising reserves indicate miners are net buyers of their own production, effectively removing supply from markets and demonstrating confidence from those with the deepest network knowledge. Declining reserves warn of accumulated supply entering distribution channels, with the potential magnitude of selling pressure proportional to reserve levels. Tracking reserves relative to mining rewards shows whether miners are accumulating new production or liquidating historical holdings. During bear markets, rising reserves often signal miner capitulation bottoms as operators who survived weak prices begin accumulating, while declining reserves may indicate distress selling. For supply analysis, high miner reserves represent potential future selling pressure or overhang, while low reserves suggest miners have already distributed holdings with less ammunition remaining for future selling.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/miner-reserve', async (req, res) => {
  try {
    const miner = req.query.miner || 'all_miner'
    const window = req.query.window || 'day'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `miner-reserve?window=${window}&miner=${miner}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      miner_reserve: d.miner_reserve,
    }))
    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
