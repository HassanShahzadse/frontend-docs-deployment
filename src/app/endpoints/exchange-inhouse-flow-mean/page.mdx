# Exchange In-House Flow (Mean)

The **Exchange In-House Flow (Mean)** metric tracks the average transaction value of Bitcoin transfers occurring within wallets of the same exchange. {{ className: 'lead' }}

This metric helps identify large internal fund reorganizations, reserve consolidations, or cold wallet management activity by exchanges.

---

## Endpoint

`GET /exchange-inhouse-flow-mean`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

```bash {title="Requesting Exchange In-House Flow (Mean) data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-inhouse-flow-mean" \
  -H "x-api-key: YOUR_API_KEY" \
  -d exchange=binance \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

The **Exchange In-House Flow (Mean)** helps identify operational fund movements within exchanges:

<Properties>
  <Property name="flow_total" type="number">
    Total BTC transferred internally within exchange wallets during the period.
  </Property>
  <Property name="flow_mean" type="number">
    Average value of internal transactions within the selected window.
  </Property>
  <Property name="transactions_count_flow" type="integer">
    Number of in-house transactions between exchange wallets.
  </Property>
  <Property name="exchange" type="string">
    Name of the exchange being analyzed.
  </Property>
</Properties>

High in-house flow values may indicate cold storage maintenance or exchange liquidity adjustments. Low activity may signal stability in internal fund structures.

  </Col>
  <Col>

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-inhouse-flow-mean', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-inhouse-flow-mean?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      flow_mean: d.flow_mean,
    }))

    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
