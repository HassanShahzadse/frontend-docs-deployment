# Exchange Inflow (Mean, MA7)

The **Exchange Inflow (Mean, MA7)** metric provides a smoothed version of the average Bitcoin inflow to exchanges over the past seven days. {{ className: 'lead' }}

This value reduces short-term noise in inflow data, highlighting broader behavioral trends among market participants depositing BTC to exchanges.

---

## Endpoint

`GET /exchange-inflow-mean-ma7`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

---

```bash {title="Requesting Exchange Inflow (Mean, MA7) data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-inflow-mean-ma7" \
  -H "x-api-key: YOUR_API_KEY" \
  -d exchange=binance \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

The **Exchange Inflow (Mean, MA7)** metric provides a rolling 7-day average of the mean BTC inflow:

<Properties>
  <Property name="inflow_total" type="number">
    Total BTC deposited to exchanges during the specified window.
  </Property>
  <Property name="inflow_top10" type="number">
    Combined inflow from the 10 largest deposit transactions.
  </Property>
  <Property name="inflow_mean" type="number">
    Average BTC deposited per transaction (unsmoothed value).
  </Property>
  <Property name="inflow_mean_ma7" type="number">
    7-day moving average of mean BTC inflow, smoothing daily fluctuations.
  </Property>
</Properties>

Rising **inflow_mean_ma7** values indicate sustained growth in average deposits — potentially signaling stronger sell-side liquidity. Declines may suggest reduced selling pressure or accumulation behavior.

  </Col>

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-inflow-mean-ma7', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-inflow-mean-ma7?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      inflow_mean_ma7: d.inflow_mean_ma7,
    }))

    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
