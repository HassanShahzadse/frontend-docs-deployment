# Exchange Inflow (Top10)

The **Exchange Inflow (Top10)** metric measures the combined Bitcoin inflow from the top 10 largest deposit transactions to exchange wallets within a given time window. {{ className: 'lead' }}

This metric is particularly useful for identifying **whale behavior** — when a few large transfers dominate inflow volume, signaling possible institutional or over-the-counter (OTC) deposits.

---

## Endpoint

`GET /exchange-inflow-top10`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

```bash {title="Requesting Exchange Inflow (Top10) data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-inflow-top10" \
  -H "x-api-key: YOUR_API_KEY" \
  -d exchange=binance \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

The **Exchange Inflow (Top10)** metric provides insights into large-scale BTC deposit behavior:

<Properties>
  <Property name="inflow_total" type="number">
    Total BTC deposited to exchange wallets during the selected window.
  </Property>
  <Property name="inflow_top10" type="number">
    Combined BTC volume of the 10 largest deposit transactions.
  </Property>
  <Property name="inflow_mean" type="number">
    Average BTC inflow per transaction.
  </Property>
  <Property name="inflow_mean_ma7" type="number">
    7-day moving average of mean BTC inflow.
  </Property>
</Properties>

High **inflow_top10** values typically indicate that whales or institutions are moving large amounts of BTC to exchanges, which may precede sell-offs or major market events.

  </Col>
  <Col>

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-inflow-top10', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-inflow-top10?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      inflow_top10: d.inflow_top10,
    }))

    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
