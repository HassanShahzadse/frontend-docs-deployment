# Exchange Outflow (Mean, MA7)

The **Exchange Outflow (Mean, MA7)** metric shows the **7-day moving average** of the average BTC transaction value leaving exchange wallets. {{ className: 'lead' }}

This moving average helps smooth daily volatility and better visualize withdrawal trends that may indicate accumulation behavior by large holders.

---

## Endpoint

`GET /exchange-outflow-mean-ma7`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

```bash {title="Requesting Exchange Outflow (Mean, MA7) data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-outflow-mean-ma7" \
  -H "x-api-key: YOUR_API_KEY" \
  -d exchange=binance \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

<Properties>
  <Property name="outflow_mean_ma7" type="number">
    7-day moving average of average BTC amount per withdrawal transaction from
    exchanges.
  </Property>
</Properties>

### Interpretation

- **Rising MA7 Outflow (↑)** → Consistent large withdrawals, often a **bullish** indicator of accumulation.
- **Falling MA7 Outflow (↓)** → Smaller withdrawals or declining activity, possibly **neutral** or **bearish** sentiment.

The moving average smooths daily anomalies caused by single large transfers, providing a clearer long-term signal.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-outflow-mean-ma7', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-outflow-mean-ma7?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      outflow_mean_ma7: d.outflow_mean_ma7,
    }))
    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
