# Exchange Inflow CDD (Coin Days Destroyed)

The **Exchange Inflow CDD** metric measures the age-weighted volume of Bitcoin flowing into exchanges — a subset of the broader Coin Days Destroyed (CDD) indicator. {{ className: 'lead' }}

It represents how long coins were held before being sent to exchanges, filtering out on-chain noise and providing a more accurate view of long-term holder (LTH) movements into exchange wallets — often a precursor to sell-side pressure.

---

## Endpoint

`GET /exchange-inflow-cdd`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

---

```bash {title="Requesting Exchange Inflow CDD data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-inflow-cdd" \
  -H "x-api-key: YOUR_API_KEY" \
  -d exchange=binance \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

**Coin Days Destroyed (CDD)** measures how long coins remained dormant before being moved. The longer coins are held, the more “coin days” they accumulate. When these aged coins are spent — especially into exchanges — it signals that long-term holders are likely taking profit or reducing exposure.

<Properties>
  <Property name="inflow_cdd" type="number">
    Total age-weighted BTC inflow (Coin Days Destroyed) entering exchange
    wallets within the specified window.
  </Property>
</Properties>

### Interpretation

- **Rising Exchange Inflow CDD** → Long-term holders (LTHs) are sending older coins to exchanges, often signaling increased sell pressure or distribution.
- **Low Exchange Inflow CDD** → Younger coins dominate exchange inflows, suggesting short-term trading or fresh accumulation.

This metric provides a more stable, filtered version of total CDD, reducing the impact of smaller, high-frequency transactions.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-inflow-cdd', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-inflow-cdd?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      inflow_cdd: d.inflow_cdd,
    }))
    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
