# Exchange Depositing Transactions

The **Exchange Depositing Transactions** metric counts the total number of Bitcoin transactions sending funds to exchange wallets. {{ className: 'lead' }}

Higher inflows of transactions often correspond to increased market activity, potential sell-offs, or profit-taking events. Conversely, declining deposit activity can indicate accumulation or lower trading pressure.

---

## Endpoint

`GET /exchange-depositing-transactions`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

---

This example retrieves the number of deposit transactions per day. Adjust parameters to specify an exchange, timeframe, or export format.

```bash {title="Requesting Exchange Depositing Transactions data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-depositing-transactions" \
  -H "x-api-key: YOUR_API_KEY" \
  -d exchange=binance \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

The **Exchange Depositing Transactions** metric helps assess short-term and long-term market activity:

<Properties>
  <Property name="transactions_count_inflow" type="number">
    Number of Bitcoin transactions depositing funds to exchange wallets.
  </Property>
  <Property name="transactions_count_outflow" type="number">
    Number of Bitcoin transactions withdrawing funds from exchange wallets.
  </Property>
  <Property name="exchange" type="string">
    Name of the exchange where the transaction activity is monitored.
  </Property>
</Properties>

High transaction counts indicate strong participation and potential sell-side pressure. Low counts suggest calm market conditions and reduced speculative trading.

  </Col>
  <Col>

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-depositing-transactions', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-depositing-transactions?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      transactions_count_inflow: d.transactions_count_inflow,
    }))

    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
