# Adjusted SOPR (aSOPR)

The **Adjusted Spent Output Profit Ratio (aSOPR)** measures the profitability of spent transaction outputs on the Bitcoin network. {{ className: 'lead' }}  
It compares the USD value of outputs when they are spent to when they were created, showing whether coins were moved at a profit or loss.

The adjusted version excludes transactions with lifetimes shorter than one hour, filtering out noise from rapid trading and self-transfers, giving a clearer view of broader market sentiment.

---

## Endpoint

`GET /adjusted-sopr`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

```bash {title="Requesting Adjusted SOPR data"}
curl -G "https://api.blocklytics.net/api/public/v1/adjusted-sopr" \
  -H "x-api-key: YOUR_API_KEY" \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

The **aSOPR** is a ratio calculated as the USD value of spent outputs at the time of spending divided by the USD value of those same outputs at creation.

A value greater than `1` indicates that coins were sold at a profit, while a value below `1` shows they were sold at a loss.  
This adjusted metric provides a smoother signal by excluding short-term noise, helping identify macro profit-taking or capitulation periods.

<Properties>
  <Property name="a_sopr" type="number">
    Adjusted Spent Output Profit Ratio. Values above `1` indicate profit; values
    below `1` indicate loss.
  </Property>
</Properties>

  </Col>
  <Col>

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/adjusted-sopr', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `adjusted-sopr?window=${window}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      a_sopr: d.a_sopr,
    }))

    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
