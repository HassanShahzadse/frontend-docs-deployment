# Exchange In-House Flow (Total)

The **Exchange In-House Flow (Total)** metric tracks the total amount of Bitcoin transferred between wallets belonging to the same exchange. {{ className: 'lead' }}

This value represents internal transfers such as hot-to-cold wallet movements, internal bookkeeping, or liquidity management. It provides insights into exchange-level activity unrelated to external user deposits or withdrawals.

---

## Endpoint

`GET /exchange-inhouse-flow-total`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

```bash {title="Requesting Exchange In-House Flow (Total) data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-inhouse-flow-total" \
  -H "x-api-key: YOUR_API_KEY" \
  -d exchange=binance \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

The **Exchange In-House Flow (Total)** provides visibility into the scale of internal fund movement within exchanges:

<Properties>
  <Property name="flow_total" type="number">
    Total Bitcoin amount transferred internally between exchange-owned wallets
    during the given period.
  </Property>
  <Property name="flow_mean" type="number">
    Average value of in-house transactions within the same timeframe.
  </Property>
  <Property name="transactions_count_flow" type="integer">
    Number of internal transactions between exchange wallets.
  </Property>
  <Property name="exchange" type="string">
    Name of the exchange for which data is collected.
  </Property>
</Properties>

Spikes in **flow_total** may indicate reserve reallocation or custodial maintenance. Low values suggest minimal internal wallet activity.

  </Col>
  <Col>

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-inhouse-flow-total', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-inhouse-flow-total?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      flow_total: d.flow_total,
    }))

    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
