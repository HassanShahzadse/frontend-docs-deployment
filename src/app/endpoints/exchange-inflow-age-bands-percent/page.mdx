# Exchange Inflow – Spent Output Age Bands (%)

The **Exchange Inflow – Spent Output Age Bands (%)** metric shows the relative percentage distribution of Bitcoin inflows to exchanges based on the age of spent outputs. {{ className: 'lead' }}

This normalized version of the age bands indicator makes it easier to compare the dominance of different holder groups over time — for example, when long-term holders (LTHs) start increasing their share of deposits, signaling potential market tops or distribution phases.

---

## Endpoint

`GET /exchange-inflow-age-bands-percent`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

---

```bash {title="Requesting Exchange Inflow Age Bands (%) data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-inflow-age-bands-percent" \

  -d window=day \
```

## Understanding the metric

<Row>
  <Col>

Each percentage-based field reflects the relative contribution of coins from different age groups to total exchange inflows on a given day.

<Properties>
  <Property name="range_0d_1d_percent" type="number">
    % of BTC inflows from coins moved within the last 24 hours.
  </Property>
  <Property name="range_1d_1w_percent" type="number">
    % of BTC inflows from coins moved within the past week.
  </Property>
  <Property name="range_1w_1m_percent" type="number">
    % of BTC inflows from coins held for 1 week – 1 month.
  </Property>
  <Property name="range_1m_3m_percent" type="number">
    % of BTC inflows from coins held for 1 – 3 months.
  </Property>
  <Property name="range_3m_6m_percent" type="number">
    % of BTC inflows from coins held for 3 – 6 months.
  </Property>
  <Property name="range_6m_12m_percent" type="number">
    % of BTC inflows from coins held for 6 – 12 months.
  </Property>
  <Property name="range_12m_18m_percent" type="number">
    % of BTC inflows from coins held for 12 – 18 months.
  </Property>
  <Property name="range_18m_2y_percent" type="number">
    % of BTC inflows from coins held between 18 months – 2 years.
  </Property>
  <Property name="range_2y_3y_percent" type="number">
    % of BTC inflows from coins aged 2 – 3 years.
  </Property>
  <Property name="range_3y_5y_percent" type="number">
    % of BTC inflows from coins aged 3 – 5 years.
  </Property>
  <Property name="range_5y_7y_percent" type="number">
    % of BTC inflows from coins aged 5 – 7 years.
  </Property>
  <Property name="range_7y_10y_percent" type="number">
    % of BTC inflows from coins aged 7 – 10 years.
  </Property>
  <Property name="range_10y_inf_percent" type="number">
    % of BTC inflows from coins older than 10 years.
  </Property>
</Properties>

When older coins (e.g., >3y) start accounting for a higher percentage of total inflows, it can indicate that long-term holders are moving their coins to exchanges — often preceding profit-taking cycles or market corrections.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-inflow-age-bands-percent', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-inflow-age-bands-percent?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      range_0d_1d_percent: d.range_0d_1d_percent,
      range_1d_1w_percent: d.range_1d_1w_percent,
      range_1w_1m_percent: d.range_1w_1m_percent,
      range_1m_3m_percent: d.range_1m_3m_percent,
      range_3m_6m_percent: d.range_3m_6m_percent,
      range_6m_12m_percent: d.range_6m_12m_percent,
      range_12m_18m_percent: d.range_12m_18m_percent,
      range_18m_2y_percent: d.range_18m_2y_percent,
      range_2y_3y_percent: d.range_2y_3y_percent,
      range_3y_5y_percent: d.range_3y_5y_percent,
      range_5y_7y_percent: d.range_5y_7y_percent,
      range_7y_10y_percent: d.range_7y_10y_percent,
      range_10y_inf_percent: d.range_10y_inf_percent,
    }))

    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
