# Exchange Reserve

The **Exchange Reserve** metric represents the **total amount of Bitcoin held in exchange wallets**. {{ className: 'lead' }}

Monitoring reserves helps identify liquidity trends and investor behavior — declining reserves usually indicate coins moving to self-custody (bullish), while increasing reserves suggest selling pressure (bearish).

---

## Endpoint

`GET /exchange-reserve`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

```bash {title="Requesting Exchange Reserve data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-reserve" \
  -H "x-api-key: YOUR_API_KEY" \
  -d exchange=binance \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

<Properties>

    Total BTC held in exchange wallets during the selected period.
  </Property>
  <Property name="reserve_usd" type="number">
    Equivalent USD value of total BTC reserves (if available).
  </Property>
</Properties>

### Interpretation

- **Falling Exchange Reserves (↓)** → Coins moving off exchanges → **bullish** (holders prefer self-custody, reduced sell pressure).
- **Rising Exchange Reserves (↑)** → Coins moving onto exchanges → **bearish** (increased potential selling activity).

When combined with inflow/outflow metrics, this indicator reveals shifts in liquidity and market sentiment.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-reserve', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-reserve?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({ date: d.date, reserve: d.reserve }))
    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
