# Exchange Outflow (Mean)

The **Exchange Outflow (Mean)** metric measures the **average value of Bitcoin transactions leaving exchange wallets** over a specified period. {{ className: 'lead' }}

This metric helps track withdrawal behavior — high average outflows may indicate large holders (whales) moving BTC off exchanges for custody, suggesting bullish sentiment.

---

## Endpoint

`GET /exchange-outflow-mean`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

---

```bash {title="Requesting Exchange Outflow (Mean) data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-outflow-mean" \
  -H "x-api-key: YOUR_API_KEY" \
  -d exchange=binance \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

<Properties>
  <Property name="outflow_mean" type="number">
    Average BTC amount per withdrawal transaction from exchange wallets during
    the selected period.
  </Property>
</Properties>

### Interpretation

- **High Average Outflow (↑)** → Large withdrawals from exchanges, possibly whale accumulation or long-term custody moves.
- **Low Average Outflow (↓)** → Smaller, retail-dominated withdrawals or lower activity.

Comparing this with inflow metrics can help detect whether market participants are accumulating or distributing BTC.

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-outflow-mean', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-outflow-mean?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      outflow_mean: d.outflow_mean,
    }))
    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
