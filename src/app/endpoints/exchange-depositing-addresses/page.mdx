# Exchange Depositing Addresses

The **Exchange Depositing Addresses** metric tracks the number of unique Bitcoin addresses depositing funds to exchange wallets. {{ className: 'lead' }}

Rising depositing activity often signals increased selling pressure or profit realization by traders, while declining activity suggests accumulation or reduced trading intensity.

---

## Endpoint

`GET /exchange-depositing-addresses`

**Base URL:**  
`https://api.blocklytics.net/api/public/v1`

---

This example retrieves the number of addresses depositing Bitcoin to exchanges. Adjust parameters to specify an exchange, timeframe, or output format.

  <Property name="to" type="string">
    End time in <code>YYYYMMDDTHHMMSS</code> or <code>YYYYMMDD</code>. For{' '}

  <Property name="limit" type="integer">

```bash {title="Requesting Exchange Depositing Addresses data"}
curl -G "https://api.blocklytics.net/api/public/v1/exchange-depositing-addresses" \
  -H "x-api-key: YOUR_API_KEY" \
  -d exchange=binance \
  -d window=day \
  -d from=20250101 \
  -d to=20250107 \
  -d limit=2
```

## Understanding the metric

<Row>
  <Col>

The **Exchange Depositing Addresses** metric helps identify the flow of Bitcoin into exchanges — a proxy for market sentiment and potential price movements:

<Properties>
  <Property name="addresses_count_inflow" type="number">
    Number of unique Bitcoin addresses depositing funds to exchange wallets.
  </Property>
  <Property name="exchange" type="string">
    Name of the exchange where deposits are tracked.
  </Property>
</Properties>

High inflows typically indicate traders preparing to sell or rebalance portfolios. Low inflows can imply reduced selling pressure and accumulation behavior.

  </Col>
  <Col>

```js {title="Node.js (Express) example"}
// Example Express route using Blocklytics
router.get('/exchange-depositing-addresses', async (req, res) => {
  try {
    const window = req.query.window || 'day'
    const exchange = req.query.exchange || 'all_exchange'
    const from = req.query.from ? `&from=${req.query.from}` : ''
    const to = req.query.to ? `&to=${req.query.to}` : ''
    const limit = req.query.limit ? `&limit=${req.query.limit}` : ''
    const format = req.query.format ? `&format=${req.query.format}` : ''

    const path = `exchange-depositing-addresses?window=${window}&exchange=${exchange}${from}${to}${limit}${format}`
    const data = await blocklyticsFetch(path)

    if (!Array.isArray(data)) {
      return res.status(500).json({ error: 'Unexpected API format', data })
    }

    const formatted = data.map((d) => ({
      date: d.date,
      addresses_count_inflow: d.addresses_count_inflow,
    }))

    res.json(formatted)
  } catch (err) {
    res.status(500).json({ error: err.message })
  }
})
```
